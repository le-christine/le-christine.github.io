Community Cares

Community Cares

(made with Java SpringBoot, React.js, PostgreSQL)

Project Community Cares aims to foster the relationship between people and government. It utilizes a NYC Open Data API that delivers information about free resources in New York City. Users customize their query based on age group and program type. From nutrition, free tax preparation, and health clinics, users are empowered to self advocate and find support they need.

Audience

Community Cares serves all New Yorkers. It may be useful for those like Jasmine* who is a high school student looking for summer employment, or Kate*, who is a new parent and seeks child care services.

Challenges

Since I used an API from NYC Open Data, I considered consuming the API data on the backend. In my initial design, I would have two microservices, one handling the API data, and the other holds users' data. This design complies to the idea of microservices and separate concerns. Ultimately I decided to fetch data on the frontend, and save user's query to ensure that they would receive accurate information.

* - Fictional user
